# IBM Cognos Analytics Dashboard – Developer Survey

This project presents interactive dashboards built using **IBM Cognos Analytics** based on the Stack Overflow Developer Survey dataset.

## Dashboards Included

* Current Technology Usage
* Future Technology Trends
* Developer Demographics
